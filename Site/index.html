Palets Game!

Bienvenue sur le site internet de Palets Game!

Palets Game est un jeu-vidéo qui imite le célèbre "Jeu du Palet". Votre mission, si vous l'acceptez, est de marquer le plus de points possible dans l'équipe adverse.

Armez-vous à l'aide de vos palets pour marquer le plus de points possible !


Palets Game! est disponible en multijoueur réseau ou sur le même ordinateur !
Jouez avec tous vos amis pour des parties interminable !
